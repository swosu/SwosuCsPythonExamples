\section{Introduction}

\begin{frame}{Learning Objectives}
\begin{itemize}
    \item Define microservices architecture.
    \item Explain how microservices differ from monolithic systems.
    \item Identify major advantages and challenges.
    \item Evaluate when microservices are a good design choice.
\end{itemize}
\end{frame}

\begin{frame}{What is Microservices Architecture?}
\begin{itemize}
    \item A software system is built as a set of small, independent services.
    \item Each service supports a specific business capability.
    \item Services communicate through APIs or messaging.
    \item Services can often be deployed and scaled independently.
\end{itemize}
\end{frame}
