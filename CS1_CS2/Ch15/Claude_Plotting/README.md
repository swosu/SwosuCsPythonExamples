# Plotting Foundations for CS2

A beginner-friendly Jupyter notebook workbook for CS2 students who have never plotted data before.

This lesson introduces the mechanics of plotting and helps students use charts to explain what data means, not just how to draw lines and bars.

## Learning Goals

- Build basic plots with clear titles, labels, legends, and grids.
- Use plots to tell a story about patterns, changes, thresholds, and comparisons.
- Choose chart types that match the question being asked.
- Compare Fibonacci algorithms by visualizing how their performance changes.

## Required Packages

- `matplotlib`
- `numpy`
- `pandas`
- `jupyter`

## How to Open the Notebook

```bash
jupyter notebook notebooks/plotting_foundations_cs2.ipynb
```

## Data Policy

This notebook uses only synthetic, hard-coded data. It does not use real student records, Canvas exports, or private files of any kind.

## Suggested Classroom Flow

1. Why plotting matters and what makes a good plot.
2. A first line plot using plain Python lists.
3. NumPy arrays for plotting and summary statistics.
4. Transforming data with a score-curving example.
5. Pandas DataFrames and bar charts.
6. Storytelling with line plots and monthly status data.
7. Scatter plots for capacity versus potential.
8. Subplots for side-by-side comparison.
9. A Fibonacci algorithm comparison story.
10. Mini-report practice and interpretation.
